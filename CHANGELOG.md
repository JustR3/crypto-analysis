# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-06

### Added
- Initial project structure
- Data fetching from exchanges via CCXT
- Local caching system for API responses
- Technical indicators (RSI, MACD, Bollinger Bands, Moving Averages, ATR, Stochastic)
- Visualization functions for price charts and indicators
- Three trading strategies (MA crossover, RSI, MACD)
- Simple backtesting framework
- Jupyter notebook for interactive analysis
- CLI interface via main.py
- Comprehensive documentation
- Test framework structure
- Configuration management

### Changed
- Reorganized project structure for better maintainability
- Moved documentation to docs/ directory
- Moved shell scripts to scripts/ directory
- Enhanced .gitignore file

### Fixed
- Removed duplicate setup script (fix_kernel.sh)
